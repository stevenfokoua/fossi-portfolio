Featured Work

Flagship Case Study

Mobile Force Verification Kit

A collaborative university project developed with INNOVATEST to create a compact verification kit for hardness testers. The system replaces bulkier calibration workflows with a portable device that can recognize load cells automatically, measure force accurately, log traceable data, and support field engineers on-site.

Problem

Hardness testers must be checked and calibrated regularly, but traditional setups are cumbersome for workshops and field service. The brief called for a compact, rechargeable, robust alternative.

System Direction

One handheld control unit, six compatible load cells, automatic cell detection, calibrated measurement logic, touchscreen interaction, and exportable audit-trail data for ISO and ASTM traceability.

Engineering Themes

Sensor interfacing, embedded data handling, HMI usability, enclosure trade-offs, field service reliability, and a product narrative that could speak to both technicians and stakeholders.

Portfolio Value

This project shows how I work across requirements, concept development, interface thinking, technical communication, and polished presentation, not just isolated engineering tasks.

Key Product Signals

  • 24-bit ADC-based measurement path for precise digitization
  • Automatic load-cell recognition through ID resistor logic
  • USB-C and microSD workflow for logging and export
  • Real-time plus peak-force display for service tasks
  • Portable form factor built for field and workshop conditions
